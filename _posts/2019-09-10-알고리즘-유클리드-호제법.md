# 유클리드 호제법
## 출처: 한국코드페어

```python3
def gcd(a, b):
    if b == 0:
        return a
    else:
        return gcd(b, a%b)

str1 = input()
arr = str1.split(' ')
a = int(arr[0])
b = int(arr[1])
			
print(gcd(a, b))
```
